Project Proposal

Title: Interactive 2D Platform for Exploring the USP Campus

Abstract: This project aims to develop a 2D platform, similar to old-school Pokémon games, simulating the experience of a new student exploring the USP campus. The application will be interactive and educational, allowing the user to learn about the buildings, services, fun facts, and common paths around the campus.
