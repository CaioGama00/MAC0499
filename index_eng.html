Project Proposal

Title: Interactive 2D Platform for Exploring the USP Campus

Abstract: This project aims to develop a 2D platform, inspired by old Pokémon games, that simulates the experience of a new student exploring the USP campus. The application will be interactive and educational, allowing users to learn about buildings, services, curiosities, and common paths on campus.
